# 📊 Sales Data Analysis using Python

## 📌 Project Overview
This project analyzes a retail sales dataset to uncover meaningful business insights using Python.

## 🛠 Tools & Technologies
- Python
- Pandas (Data Analysis)
- Matplotlib (Data Visualization)

## 📂 Dataset
Superstore sales dataset containing order details, sales, region, and category information.

## 📈 Key Insights
- West region generates the highest sales among all regions
- Technology category is the top-performing segment
- Certain regions show comparatively lower sales performance
